# Forum (Spring Boot App)

#### Description:
Repository contains implementation simple model of forum which demonstrate working with Spring Boot. 
                         
#### Technologies:
- Java 11
- Spring Boot(Data, Security, Test)
- PostgreSQL
- Maven
- Liquibase
- JSP, HTML, Bootstrap 5
